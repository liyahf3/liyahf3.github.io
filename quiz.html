Celebrity Quiz

The Celebrity Quiz Project is a simple Javascript based project that was built on codehs meant to test the user on a celebrity of my choosing. Similar to how buzzfeed quizzes work with answer choices and scoring. There was some HTMl used in this project, but it was mainly Javscript. As you open the link to the project, you will be prompted by your browser to answer several questions about Billie Eilish and the after you will be given a score. I created all of the code that you see in the codespace and was responsible for the entire project alone. One of the hardest parts of this project was making sure that the scoring was working properly and the answers that were typed were ones that could be accepted. I was running my code by peers and looking at online resources to make sure these things were correct and were being done the proper way.
