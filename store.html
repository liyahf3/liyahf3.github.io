Virtual Store Shopping

The Virtual Store Shopping is a project that I recently did and it's basically a simpler version of my e commerce webiste. I used simple javascript to ask the user for budget and gave them items with prices to shop from. This project was also built on CodeHs and I was responsible for all the functions of the project. One of my biggest challenges with this project was making sure that the case sensitivity was working properly. To make sure that the user wasn't purchasing something that didn't exist, we had to make sure that whatever they typed in actually was present in the array. Making sure that it was case sensitive and correctly working within certain parameters was very important to the project. Overall, this is a simple user shopping site than can be made into something a little more advanced with more time.
