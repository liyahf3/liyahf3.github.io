The E-commerce Website is a project I created in 2025 and I created it so people could "shop" my collection of t-shirts. My teacher helped us create some of the base code, but I was in charge of everything else. I chose the images, designed the website, built and formatted the code. I utilized HTML, CSS, and Javascript. Upon clicking the link on the home page, users can see the collection of t shirts, add any t-shirts to cart, enter their information, and then be led to a purchase confirmation page. The hardest part about this project was making sure that all of the buttons led to the corect page and correpsonded properly, but I was able to get over this hump with online coding resources to fact check my work.
